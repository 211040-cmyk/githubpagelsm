PROTOTIPO V1.0

Traductor de Lengua de Señas Mexicana

Una herramienta tecnológica diseñada para cerrar la brecha de comunicación mediante el dactileo y deletreo automatizado de señas.

Sobre el Proyecto

Este proyecto es un **prototipo de sistema inteligente** enfocado en la traducción de la Lengua de Señas Mexicana (LMS). Su función principal actualmente es el **deletreo de palabras letra por letra (dactilología)** a través del reconocimiento y procesamiento visual de las señas de las manos.

Estado actual: El sistema se encuentra en fase de desarrollo temprano. Reconoce el alfabeto básico (A-Z) de forma estática para construir palabras de manera progresiva.

¿Cómo funciona el sistema?

El prototipo utiliza procesamiento de imágenes y modelos de aprendizaje para realizar la traducción en tres pasos clave:

1

Captura de Video

El usuario realiza la seña de la letra frente a una cámara web o sensor en tiempo real.

2

Procesamiento

El algoritmo analiza los puntos clave de la mano para identificar la letra correspondiente.

3

Texto en Pantalla

La letra identificada se traduce a texto y se suma para formar palabras completas.

Objetivo e Impacto Social

La falta de conocimiento de la LMS por parte de la población oyente crea barreras diarias para la comunidad sorda. El objetivo de este traductor es servir como una herramienta de apoyo didáctico e inclusión social, permitiendo una comunicación básica, fluida y accesible para cualquier persona sin importar si conoce o no la lengua de señas.
